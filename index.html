dietaryindex

An open-source toolkit for computing dietary pattern indexes.

This website accompanies the dietaryindex repository. The project was originally created by Jiada (James) Zhan and colleagues. The University of Florida is extending the software with a modernized interface and additional Python utilities. The goal is to provide simple online calculators for the Dietary Inflammatory Index (DII), MIND diet score and Healthy Eating Index (HEI).

Python Examples

pip install -e .
from dietaryindex_py import dii_score, mind_score, hei_score
